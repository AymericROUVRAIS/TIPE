## Contenu

On retrouve ici le code python utilisé lors de la phase de simulation. Ils n'ont pas été utilisé, car je n'arrivais pas à faire fonctionner correctement le logiciel de CFD du Dr Lehmann. J'ai donc opté pour la simulation en ligne avec SimScale.
Le code auto_cfd.py a pour but d'automatiser le lancement des simulations pour faire plus facilement un maximum de simulations.
Le second, post_process.py, a lui été utilisé pour tracer les graphes et traiter les résultats du logiciel de CFD.

Au final, <a href="https://github.com/AymericROUVRAIS/TIPE/tree/PostProcess">PostProcess</a> est la branche qui correspond à du code utilisé lors de mon TIPE.

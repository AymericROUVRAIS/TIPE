## Contenu

Les code python .py correspondent à différement traitement des données expérimentales, ou théoriques.
- generate.py génère artificiellement des données pour tester les fichiers de génération de graph
- graph_theorie.py trace le graphe des résultats théoriques obtenu par CFD
- post_process_exp.py traite les données expérimentales des fichiers de valeurs expérimentales
- post_process.py corrige des erreurs d'algorithmie de post_process.py
- post_process-2.py est la version plus paufiné et mieux présenté ainsi que mieux documenté que post_process.py
- traitement_theorique.py permet de régler des problèmes de données mal écrite dans un fichier de données.
Les fichiers post_process-2.py et graph_theorie.py sont ceux utilisés pour générer les graphes que l'on retrouve dans la présentation du TIPE.

Les fichiers .txt et .csv sont les mêmes, seul l'extension du fichier diffère.
